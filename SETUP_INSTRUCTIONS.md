# Setup Instructions

## Initial Setup

1. **Install Dependencies**
   ```bash
   npm install
   ```
   
   Note: You may see TypeScript errors before running this command. These will be resolved once dependencies are installed.

2. **Run Storybook**
   ```bash
   npm run storybook
   ```
   
   Storybook will open at `http://localhost:6006`

3. **Build Storybook for Deployment**
   ```bash
   npm run build-storybook
   ```

## Development

- **Run Dev Server**: `npm run dev`
- **Build for Production**: `npm run build`
- **Preview Build**: `npm run preview`
